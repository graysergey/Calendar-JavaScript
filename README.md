## Calendar of events

1. Clone:

    ```
    git@github.com:graysergey/Calendar-JavaScript.git
    ```

2. Install prerequisites:

    ```
    npm i --save-dev
    ```

* Start dev server with hot reload:

    ```npm run start```
    
* Test linters:

    ```npm run test```

* Build and optimize images for production:

    ```npm run build```
